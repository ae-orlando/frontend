# Snippet Finder

A small beginner JavaScript project that practices several useful string methods:

- `includes()` to search notes by keyword
- `slice()` to show a short preview of each note
- `trim()` to remove extra spaces from the search
- `toLowerCase()` to make searching case-insensitive
- `toUpperCase()` to show the text in all caps

## What it does

This app displays a list of JavaScript learning notes. As you type in the search box, it filters the notes in real time. Each note only shows a shortened preview of the full text.
It also shows your current input in its original, trimmed, lowercase, and uppercase forms so you can see each method in action.

## Why this is good practice

- You work with arrays of objects
- You practice DOM selection and event listeners
- You use `filter()`, `includes()`, `slice()`, `trim()`, `toLowerCase()`, and `toUpperCase()` together
- You end up with a project that is simple but still nice enough to share on GitHub

## Files

Open `index.html` in your browser.
